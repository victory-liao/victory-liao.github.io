Comparison of Methods of Artifacts Removal

Regression Methods

Regression methods often assume that the scal potential is a linear combination of brain and other potentials(EOGгҖҒECGгҖҒEMG).By subtracting propagated EOG/ECG/EMG from EEG recordings,EEG signals can be recovered.

Regression can also be done in frequency domain based on the concept that subtraction in the frequency domain is equivalent to filtering in the time domain.By eliminating spectral estimates of EOG/EMG/ECG from EEG recordings,it is possible to recover the non-contaminated EEG.

Disadvantages: Both types of regression methods are off-line and rely on EOG/ECG/EMG recordings,which are however,not always available.

WebжңҚеҠЎеҷЁ

WebжңҚеҠЎеҷЁжҳҜеҸҜд»Ҙеҗ‘еҸ‘еҮәиҜ·жұӮзҡ„жөҸи§ҲеҷЁжҸҗдҫӣж–ҮжЎЈзҡ„зЁӢеәҸгҖӮ

Code:

# coding:utf-8
import socket
import re

from multiprocessing import Process

# и®ҫзҪ®йқҷжҖҒж–Үд»¶ж №зӣ®еҪ•
HTML_ROOT_DIR = "./html"


def handle_client(client_socket):
    """еӨ„зҗҶе®ўжҲ·з«ҜиҜ·жұӮ"""
    # иҺ·еҸ–е®ўжҲ·з«ҜиҜ·жұӮж•°жҚ®
    request_data = client_socket.recv(1024)
    print("request data:",request_data)
    request_lines = request_data.splitlines()
    for line in request_lines:
        print(line)

    # и§ЈжһҗиҜ·жұӮжҠҘж–Ү
    # 'GET / HTTP/1.1'
    request_start_line = request_lines[0]
    #жҸҗеҸ–з”ЁжҲ·иҜ·жұӮзҡ„ж–Үд»¶еҗҚ
    file_name = re.match(r"\w+ +(/[^ ]*) ",request_start_line.decode("utf-8")).group(1)

    if "/" == file_name:
        file_name = "/index.html"
    # жү“ејҖж–Үд»¶пјҢиҜ»еҸ–еҶ…е®№
    try:
        file = open(HTML_ROOT_DIR + file_name,"rb")
    except IOError:
        response_start_line = "HTTP/1.1 404 Not Found\r\n"
        response_headers = "Server: My server\r\n"
        response_body = "the file is not found"
    else:
        file_data = file.read()
        file.close()

        # жһ„йҖ е“Қеә”ж•°жҚ®
        response_start_line = "HTTP/1.1 200 OK\r\n"
        response_headers = "Server: My server\r\n"
        response_body = file_data.decode("utf-8")

    response = response_start_line + response_headers + "\r\n" + response_body
    print("response:",response)

    # еҗ‘е®ўжҲ·з«Ҝиҝ”еӣһе“Қеә”ж•°жҚ®
    client_socket.send(bytes(response,"utf-8"))

    #е…ій—ӯе®ўжҲ·з«Ҝй“ҫжҺҘ
    client_socket.close()

def main():
    server_socket = socket.socket(socket.AF_INET,socket.SOCK_STREAM)
    server_socket.setsockopt(socket.SOL_SOCKET,socket.SO_REUSEADDR,1)
    #SOL_SOCKET:и®ҫзҪ®йҖүйЎ№зә§еҲ«дёәSOCKET
    #SO_REUSEADDR:1
    server_socket.bind(("",7788))
    server_socket.listen(128)

    while True:
        client_socket,client_address= server_socket.accept()
        print("[%s,%s]з”ЁжҲ·иҝһжҺҘдёҠдәҶ"%(client_address))
        handle_client_process = Process(target=handle_client,args=(client_socket,))
        handle_client_process.start()
        client_socket.close()

if __name__ == "__main__":
    main()
