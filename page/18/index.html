python | 匹配第n个分组的内容

Posted on 2021-08-28 Edited on 2021-10-27 In python , 内置模块 , re

\1…\9

\1…\9用来匹配与第n(1~9)个分组的内容，必须与()配合使用
例：在以下代码段中\2表示匹配第2个分组（一个括号代表一个分组）的内容，即\2匹配”world”字符串

import re
string = "helloworld world"
pattern = r'^(\w+)(world) \2$'
print(re.search(pattern, string))
